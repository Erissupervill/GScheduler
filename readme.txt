HOW TO RUN
step 1: Install python and xampp to device
step 2: Place Gscheduler project in xampp htdocs folder
step 3: under command prompt go to GSCHEDULER root directory and type py -3 -m venv .venv
step 4: activate .venv\Scripts\activate
step 5: once step 3 is done install flask using pip install Flask 


FRAMEWORK TYPE MVC(MODEL, VIEW, CONTROLLER)
